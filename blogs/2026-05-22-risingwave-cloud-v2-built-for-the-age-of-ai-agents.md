RisingWave Cloud V2 is a complete rebuild of the control plane designed for AI agent integration, featuring a redesigned console, a new command-line interface (rwc), and native support for AI coding agents through MCP and Agent Skills. The streaming engine itself remains unchanged, maintaining PostgreSQL compatibility and Apache 2.0 licensing.
