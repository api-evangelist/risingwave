RisingWave Cloud V2 launches in public preview as a ground-up rewrite designed for the agentic era. The release includes a redesigned console, a new CLI tool called rwc, and native AI agent support through the RisingWave Skill and MCP server, enabling developers to operate data systems across browsers, terminals, and AI coding agents.
