The RisingWave Cloud CLI (rwc) is a command-line interface that enables programmatic management of RisingWave Cloud infrastructure without requiring a browser. It supports various deployment models and integrates with AI coding assistants through Agent Skills and an MCP server for enhanced development workflows.
