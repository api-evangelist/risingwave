RisingWave Agent Skills is an open-source toolkit that teaches AI coding agents how to build stream processing pipelines correctly. It provides a core reference and a 14-rule best practices guide covering patterns like CDC ingestion, time-windowed aggregations, and performance optimization for the Source-to-Materialized View-to-Sink architecture.
